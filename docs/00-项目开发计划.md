## 风险与应对

| 风险 | 概率 | 影响 | 应对措施 |
|------|------|------|----------|
| LLM API 不稳定 | 中 | 高 | 告警文案预置 fallback 模板 |
| 预测模型效果不达标 | 低 | 中 | Prophet 基线兜底，LSTM 作为加分项 |
| 前后端联调进度滞后 | 中 | 中 | Day 4 就打通第一个 API，降低联调风险 |
| 某个成员请假 | 低 | 中 | 核心模块有文档 + 交叉备份 |
| Docker 环境问题 | 中 | 低 | 本地 Maven + JDK 直接运行作为 fallback |
