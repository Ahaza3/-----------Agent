|:-----|:-----|
| **用例 ID** | E2E-S2-001 |

| **预期结果** | 1. 负荷超限 → ThresholdDetector 判定 RED <br> 2. AlertTemplate 生成文案 → alert_event 入库 <br> 3. WebSocket `/topic/alerts` 推送告警 <br> 4. 前端告警卡片红色闪烁 <br> 5. 全链路延迟 < 5s |
