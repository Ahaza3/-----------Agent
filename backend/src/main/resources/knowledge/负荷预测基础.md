# 电力负荷预测基础
> 分类：LOAD_FORECAST

## 负荷预测的含义
电力负荷预测是根据历史负荷、天气、日期类型、小时、星期和节假日等信息，对未来某一时间段的用电负荷进行估计。短期负荷预测通常关注未来数小时到数天，常见输出包括预测曲线、预测区间和误差指标。

## 影响因素
常见影响因素包括历史负荷的周期性、气温和湿度、工作日与节假日差异、早晚峰谷规律以及突发事件。气温升高时，制冷负荷可能增加；寒潮期间，采暖负荷可能增加。实际分析时应结合区域、季节和用户结构判断，不能仅凭单一因素下结论。

## 常见评价指标
MAPE 用于衡量预测值与实际值的相对误差，RMSE 对较大的误差更敏感。指标应在相同数据集、相同预测范围和相同口径下比较。样本中实际负荷接近零时，MAPE 可能失真，应同时观察 MAE 或 RMSE。

## 使用注意
预测结果是辅助研判信息，不等同于真实调度指令。实际运行中应同时核对最新测量值、设备状态、天气变化和调度计划，并关注预测区间而不是只看单一预测点。
